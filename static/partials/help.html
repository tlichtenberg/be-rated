The default Browse Ratings view shows a random selection of some of the most recently rated items
You can use Preset Searches by selecting their radio button and clicking 'Search'
Or you can use the Filters to make your own search
Select the Use Filters radio button, then add or more filters
Select the "ratedType" filter, and type "Person", "Product", "Business", "Place" or "Other" into the value field
If you select the "sortBy" filter, type "High", "Low", "Name" or "Num" into the value field
If you select the "limit" filter, type a number into the value field to restrict the number of results displayed (or 0 to show all)
You can chain filters together to form more complex searches
For example, "ratedName > Baa" plus "ratedName < Bac" plus "ratedType = Product" plus "sortBy = Name"
In cases of filter conflicts, the last one added wins
